# TitleCheck Product Requirements Document **Brand Name:** TitleCheck **Tagline:** "What's really behind that address?" **Positioning:** Professional title intelligence and property records service. ## Visual Identity - **Primary Color:** Deep Slate Blue (#1e3a5f) - **Accent/Caution:** Amber (#b45309) - **Success:** Emerald (#059669) - **Alert:** Red (#dc2626) - **Typography:** Headlines (Libre Baskerville), Body/UI (Inter) ## Core Features 1. **Dual-Audience Landing Page:** Targeting buyers and sellers separately. 2. **Property Lookup:** Address search with autocomplete. 3. **Property Report:** Comprehensive dashboard with a Title Health Score and categorized record cards (Ownership, Liens, Foreclosure). 4. **Distress Entry Points:** Specific paths for probate, foreclosure, etc. 5. **Educational Hub:** Jargon-free guides on title issues. ## Technical Goals - Built for trust and professional authority. - Clear status indicators based on official records. - Integration with property data APIs (ATTOM).
